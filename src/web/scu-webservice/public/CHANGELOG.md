## 15.12.2025 - Version 1.0.0:
### Neue Funktionen
- Erste stabile Version der SCU Badminton Web-App
- Benutzeroberfläche mit Menü für:
  - Spieler/innen verwalten
  - Spiele verwalten
  - Pyramide / Rangliste
- Basic Auth / Access List für Admin- und User-Nutzergruppe
- PWA-Funktionalität implementiert (Installierbar auf Android & iOS)

### Hinweise zur Installation
- **Android** (empfohlener Browser: Chrome)
  - Menü über das Symbol mit den **drei Punkten** oben rechts öffnen.  
  - **"Zum Startbildschirm hinzufügen"** auswählen und im Popup **"Installieren"** bestätigen.  
  - App erscheint in der App-Übersicht und/oder auf dem Startbildschirm.
- **iOS** (empfohlener Browser: Safari) 
  - **"Teilen"**-Schaltfläche antippen.
  - **"Zum Home-Bildschirm hinzufügen"** auswählen. 
  - Oben rechts auf **"Hinzufügen"** tippen.
  - App erscheint auf dem Startbildschirm.
